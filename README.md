# Blog

Personal blog repository where I explore topics I find interesting — with a goal of one post per month.

Each folder contains the entire replication package for each blog post. Posts are meant to be reproducible: clone the repo, install dependencies, and run the scripts.

## Structure

Each post folder typically contains:
```
post_name/
├── script/       # Python scripts and job files
├── data/         # Intermediate data (not tracked in git)
└── output/       # Final outputs (stats, figures, etc.)
```
